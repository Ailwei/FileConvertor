# File Conversion Service

This is a File Conversion Service application built using the MERN stack (MongoDB, Express.js, React, Node.js). The application allows users to convert files effortlessly with different subscription plans. 

## Features

- User Authentication
  - Login
  - Register
  - Forgot Password
  - Reset Password
- Subscription Plans
  - Free Trial
  - Basic
  - Premium
- File Conversion Services
  - Video
  - Document
  - Image
  - Audio
- User Dashboard
- Checkout Process

## Technologies Used

- MongoDB
- Express.js
- React
- Node.js
- Axios
- React Router DOM
- Selemiun for end to end testing
-Stripe for payment
